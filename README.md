# Swiggy Sales Analysis Project

## 📌 Project Overview

This project analyzes Swiggy food delivery data using Python, SQL, Microsoft Excel, Power BI and Microsoft Fabric to generate business insights related to sales performance, customer behavior, restaurant trends and delivery operations.

The project covers **data cleaning, exploratory analysis, SQL querying, visualization, and dashboard development** through an end-to-end analytics workflow.

---

## 🎯 Objectives

* Analyze sales and order trends
* Understand customer ratings and food preferences
* Identify top-performing restaurants and dishes
* Track state-wise and city-wise sales performance
* Generate actionable business insights using dashboards and analytics
